Acesse o diretorio do hsql (source\lib\hsql)
execute o seguinte comando
java -cp .;hsqldb.jar org.hsqldb.Server -database.0 file:data/agendaexames -dbname.0 agendaexames

Crie uma conexao no Netbeans selecionando o hsqldb.jar e colocando a seguinte linha em URL JDBC:
jdbc:hsqldb:hsql://localhost/agendaexames

Marque no Information_Schema


Abrir o banco modo grafico
java -cp .;hsqldb.jar org.hsqldb.util.DatabaseManager
